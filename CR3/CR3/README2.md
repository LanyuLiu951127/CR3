# CR3 相同照片檢查系統 (CR3 Duplicate Photo Checker)

這是一個 Python 桌面應用程式，旨在幫助攝影師快速找出指定資料夾中的 Canon CR3 格式重複照片，以簡化篩選和整理流程。

---

## 專案介紹

在攝影工作流程中，攝影師經常會產生大量 RAW 格式檔案。本工具專為 CR3 檔案設計，透過分析檔案內容，找出完全相同的照片，並以群組方式呈現在圖形化介面中，讓使用者能一目了然，輕鬆管理檔案。

此專案符合 `doc/school/quit-stu.md` 中定義的 Python 期末專題要求。

---

## 主要功能

*   **圖形化操作介面**：提供簡單直覺的桌面應用程式介面。
*   **選擇資料夾**：使用者可輕鬆選擇要進行檢查的資料夾。
*   **自動化檢查**：點擊按鈕即可自動分析資料夾內的所有 CR3 檔案。
*   **重複照片分組**：將找到的相同照片以群組方式並排顯示，方便比對。
*   **結果快取**：初次分析後會在本機產生快取檔案，加速後續的重複檢查。
*   **跨平台支援**：基於 Python 開發，可在 Windows, macOS, Linux 上執行。

---

## 環境需求

*   Python 3.6+

---

## 安裝與執行

1.  **複製專案**
    ```bash
    git clone <your-repo-url>
    cd CR3-Duplicate-Photo-Checker
    ```

2.  **安裝依賴套件**
    本專案需要 `Pillow` 函式庫來處理圖片。
    ```bash
    pip install Pillow
    ```

3.  **執行應用程式**
    在專案根目錄下，執行主程式：
    ```bash
    python main.py
    ```
    *備註：主程式檔名可依據實際開發情況命名，例如 `app.py` 或 `run.py`。*

---

## 使用流程

1.  啟動程式後，點擊「選擇資料夾」按鈕。
2.  在跳出的視窗中，選擇您存放 CR3 檔案的資料夾。
3.  點擊「開始檢查」按鈕。
4.  程式會開始分析，並在介面上顯示進度。
5.  分析完成後，介面會顯示所有重複的照片群組。若無重複，則會顯示提示訊息。

Requirement Specification (REQ) 範例
1. 專題名稱 / Project Title / 프로젝트명
   校園社團網站 (Campus Club Website)
2. 專案目的 / Purpose / 목적
   建立一個校園社團網站,讓學生能查詢活動、公告與成員資訊。
3. 主要使用者 / Target Users / 주요 사용자
   學生(瀏覽公告與報名活動)
   幹部(發布內容與管理會員)
4. 網站主要功能 / Key Features / 주요 기능
   分類
   功能描述
   優先級
   首頁
   顯示社團簡介與最新公告
   Must
   活動頁
   報名與活動清單
   Must
   成員頁
   顯示幹部與成員資訊
   Should
   聯絡頁
   提供聯絡表單
   Could
5. 成功指標 / Success Metrics / 성공 지표
   網站能正常顯示於電腦與手機瀏覽器
   每月更新至少 3 則公告
   學生可在線上成功報名活動
6. 非範圍內容 / Out of Scope / 범위 밖
   不含登入系統與金流功能
   不包含手機App
